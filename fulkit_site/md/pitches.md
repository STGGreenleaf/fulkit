# Pitches — The Fülkit Copy Safe

## How to use this file
- Every pitch has a **slug** (e.g. `memory-contrast`) for quick reference
- `[!]` = accuracy flag — don't use as-is, needs update before publishing
- `[planned]` = feature not yet shipped — don't present as live
- Pitches are raw ingredients. Mix, adapt, resize for any surface.
